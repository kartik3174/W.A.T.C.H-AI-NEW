# W.A.T.C.H – Wildlife AI Tracking & Conservation Hub

## 🌍 Project Overview
W.A.T.C.H (Wildlife AI Tracking & Conservation Hub) is an AI and IoT based system designed to improve wildlife monitoring and conservation.  
The system collects real-time data from sensors and smart collars attached to animals and sends the data to a cloud dashboard for monitoring and analysis.

This project helps forest authorities detect abnormal animal behavior, monitor health conditions, and track wildlife movement.

---

## 🎯 Objectives
- Track wildlife location in real time
- Monitor animal health and behavior
- Detect abnormal activity or distress
- Provide alerts to forest authorities
- Support wildlife conservation efforts

---

## 🧠 Technologies Used
- **ESP32 (IoT Microcontroller)**
- **GPS Module (NEO-6M)**
- **MAX30102 Heart Rate Sensor**
- **MPU6050 Motion Sensor**
- **DS18B20 Temperature Sensor**
- **Next.js / React Dashboard**
- **Firebase Cloud Database**
- **TensorFlow.js (AI processing)**

---

## ⚙️ Hardware Components
| Component | Purpose |
|-----------|--------|
ESP32 | Main controller |
NEO-6M GPS | Animal location tracking |
MAX30102 | Heart rate monitoring |
MPU6050 | Motion and behavior detection |
DS18B20 | Body temperature monitoring |
Li-Po Battery | Portable power supply |
TP4056 Charger | Safe battery charging |
MT3608 Boost Converter | Voltage regulation |

---

## 🖥️ System Architecture
The system consists of three main layers:

1. **Data Collection Layer**
   - Sensors attached to smart collar

2. **Processing Layer**
   - ESP32 processes sensor data

3. **Cloud Monitoring Layer**
   - Data transmitted to cloud dashboard

---

## 🚀 Features
- Real-time wildlife monitoring
- Health and movement analysis
- Cloud data visualization
- Alert generation for abnormal behavior
- Scalable IoT architecture

---

## 🧪 Prototype Demonstration
The working prototype includes a wearable smart collar connected to an ESP32 microcontroller.  
Sensor data is transmitted to a web dashboard where wildlife monitoring data can be visualized in real time.
